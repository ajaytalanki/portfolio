Custom Linux Shell

This C shell is a command-line interface application that replicates the functionality of a standard Linux shell. It provides users with a familiar environment for executing Linux commands, performing file operations, and managing processes. This project implements the principles of operating systems while providing a practical tool for users who work in a Unix-like environment.

Playlist Downloader

This Python program automates the process of downloading all the tracks of a given Spotify playlist as MP3 files. This project leverages the Spotipy and Pytube APIs to interact with the Spotify and YouTube platforms respectively. The program utilizes multithreading to maximize efficiency by concurrently downloading multiple tracks from a playlist, making it valuable for users who want to enjoy their favorite Spotify playlist offline.

Game Sentiment Analysis

This sentiment analysis project is designed to provide game developers and gamers with a summarized analysis of Steam game reviews. This python project utilizes the Steamworks API, RoBERTa natural language processing model, and multithreading to efficiently analyze a sample of reviews for a given Steam app ID.

FISC CPU

The CPU Simulator showcases the ideas of computer architecture by bridging the gap between assembly code and machine code, while executing these instructions in a CPU-like manner. The program is valuable for users seeking to understand how a CPU processes instructions, experiment with custom assembly architectures, or analyze assembly code execution.

Text Editor

This application is a versatile and user-friendly text editing tool developed using Python's Tkinter library. This project combines the simplicity of a standard text editor, with advanced editing capabilities to cater to a wide range of users including developers, writers, and general users. Its rich feature set, including undo/redo, cut/copy/paste, find, replace, and file management operations, makes it an essential tool for various text-related tasks.

Battleship with AI's

A C++ BattleShip game that allows for three game modes: player vs player, player vs AI, or AI vs AI. The game offers three different AI's that each have unique special attacks. This project incorporates fundamental Object Oriented Programming concepts such inheritance and polymorphism, and uses C++ smart pointers to automatically deallocate memory. Lastly, the game incorporates key data structures and custom algorithms for special attacks.
