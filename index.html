FISC CPU

A C++ project that mimicks the process of a finite instruction set computer. This CPU contains four 8-bit registers and can process four instructions: ADD,AND,NOT,BNZ. This project is split into an assembler which transforms assembly code into hex machine code, and a simulator which reads this hex machine code and processes instructions.
